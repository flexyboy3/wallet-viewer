# Wallet Viewer

A sleek crypto wallet dashboard to track ETH and BTC transactions.

## Features
- Balance summary for ETH and BTC
- Transaction history with green and red indicators
- Filter by coin and transaction type
- Sort by date and amount
- Search transactions
- Send crypto form

## Built With
- HTML
- CSS
- JavaScript

## How to Run
1. Clone or download the repository
2. Open the folder in VS Code
3. Install Live Server extension
4. Click Go Live
